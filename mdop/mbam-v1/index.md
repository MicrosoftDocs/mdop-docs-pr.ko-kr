# Microsoft BitLocker 관리 및 모니터링 1 관리자 가이드

Microsoft BitLocker 관리 및 모니터링 (MBAM)은 BitLocker 드라이브 암호화를 관리 하는 데 사용할 수 있는 간단한 관리 인터페이스를 제공 합니다. MBAM을 사용 하는 경우 엔터프라이즈에 적합 한 BitLocker 암호화 정책 옵션을 선택한 다음 해당 정책으로 클라이언트 준수를 모니터링 하는 데 사용할 수 있습니다. 개별 컴퓨터 및 전체 엔터프라이즈의 암호화 상태에 대해 보고할 수도 있습니다. 또한 사용자가 PIN 또는 암호를 잊어버린 경우 또는 해당 BIOS 또는 부팅 레코드가 변경 될 때 복구 키 정보에 액세스할 수 있습니다.
