# DaRT 복구 이미지 마법사를 사용하여 복구 이미지를 만드는 방법


Microsoft 진단 및 복구 도구 집합 (DaRT) 7에는 Windows에서 부팅 가능한 국제 표준화 (ISO) 이미지를 만드는 데 사용 되는 **DaRT 복구 이미지 마법사** 가 포함 되어 있습니다. ISO 이미지는 CD의 원시 콘텐츠를 나타내는 파일입니다.

**DaRT 복구 이미지 마법사** 에는 다음 정보가 필요 합니다.

-   **부팅 이미지**° DaRT 복구 이미지를 만드는 데 필요한 WINDOWS 7 DVD 또는 windows 7 원본 파일의 경로를 제공 해야 합니다.

-   **도구 선택**스타일 DaRT 복구 이미지에 포함할 도구를 선택할 수 있습니다.

-   **원격 연결**°-DaRT 복구 이미지에 헬프데스크 및 최종 사용자 컴퓨터 간의 원격 연결을 설정 하는 기능을 포함할지 여부를 선택할 수 있습니다.

-   **Windows 용 디버깅 도구**windows 용 디버깅 도구 위치를 제공 하 라는 메시지가 표시 됩니다.

-   **독립 실행형 시스템 Sweeper에 대 한 정의**복구 이미지를 만들 때 최신 정의를 다운로드할지 여부를 결정 하거나 나중에 정의를 다운로드할 수 있습니다.

-   **드라이버**를 통해 ISO 이미지에 드라이버를 추가할지 묻는 메시지가 표시 됩니다.

-   **추가 파일**, 문제를 진단 하는 데 도움이 되는 파일을 ISO 이미지에 추가할 수 있습니다.

-   **Iso 이미지 위치**° ° iso 이미지를 배치할 위치를 지정 하 라는 메시지가 표시 됩니다.

-   Cd **/Dvd 드라이브**° ° cd 또는 dvd 드라이브를 사용 하 여 CD 또는 dvd를 굽는 지 여부를 지정 하 라는 메시지가 표시 됩니다.

**참고**  ISO 이미지 크기는 **DaRT 복구 이미지 마법사**에서 선택한 도구에 따라 다를 수 있습니다.

 

## DaRT 복구 이미지 마법사를 사용 하 여 복구 이미지를 만들려면


**Dart 복구 이미지 마법사** 를 사용 하 여 dart 복구 이미지를 만들려면 다음 지침을 따르세요.

### DaRT 복구 이미지에 포함할 도구를 선택 하려면

**DaRT 복구 이미지 마법사** 는 **도구 선택** 대화 상자를 표시 합니다. 도구를 강조 표시 한 다음 **사용** 또는 **사용 안 함** 단추를 클릭 하 여 DaRT 복구 이미지에 포함할 도구 목록에서 도구를 선택 하거나 제거할 수 있습니다.

복구 이미지에 포함 하려는 모든 도구를 선택 했으면 **다음**을 클릭 합니다.

### 원격 연결을 허용 하는 옵션을 추가 하려면

**원격 연결 허용** 확인란을 선택 하 여 **진단 및 복구 도구 집합** 창에서 헬프데스크 에이전트와 최종 사용자 컴퓨터 간에 원격 연결을 설정 하는 옵션을 제공할 수 있습니다. 헬프데스크 에이전트는 원격 연결을 설정 하 고 나면 원격 위치에서 최종 사용자 컴퓨터에 있는 DaRT 도구를 실행할 수 있습니다.

**포트 번호 지정** 확인란을 선택 하 여 원격 연결을 설정할 때 사용 되는 특정 포트 번호를 입력할 수 있습니다. 1에서 65535 사이의 포트 번호를 지정할 수 있습니다. 충돌의 가능성을 최소화 하기 위해 포트 번호를 1024 이상으로 하는 것이 좋습니다.

원격 연결을 설정할 때 최종 사용자가 받는 사용자 지정 메시지를 만들 수도 있습니다. 메시지는 최대 2048 자를 초과할 수 있습니다.

DaRT 도구를 원격으로 실행 하는 방법에 대 한 자세한 내용은 [Dart 복구 이미지를 사용 하 여 원격 컴퓨터를 복구 하는 방법을](how-to-recover-remote-computers-using-the-dart-recovery-image-dart-7.md)참조 하세요.

### Windows 용 디버깅 도구를 DaRT 복구 이미지에 추가 하려면

**DaRT 복구 이미지 마법사**의 **크래시 분석기** 대화 상자에서 Windows 용 디버깅 도구 위치를 지정 하 라는 메시지가 표시 됩니다. 도구의 복사본이 없는 경우 Microsoft에서 다운로드할 수 있습니다. 다운로드 페이지에 대 한 다음 링크는 마법사에 제공 됩니다. [Windows 용 디버깅 도구를 다운로드 하 고 설치](https://go.microsoft.com/fwlink/?LinkId=99934)합니다.

**DaRT 복구 이미지 마법사**를 실행 하는 컴퓨터에서 디버깅 도구의 위치를 지정 하거나 대상 컴퓨터에 있는 도구를 사용 하도록 결정할 수 있습니다. 다른 컴퓨터의 복사본을 사용 하기로 결정 한 경우에는 충돌을 진단 하는 각 컴퓨터에 도구가 설치 되어 있는지 확인 해야 합니다.

**참고**  ISO 이미지에 **충돌 분석기** 를 포함 하는 경우 Windows 용 디버깅 도구도 포함 하는 것이 좋습니다.

 

Windows 용 디버깅 도구를 추가 하려면 다음 단계를 따르세요.

1.  ) 하이퍼링크를 클릭 하 여 Windows 용 디버깅 도구를 다운로드 합니다.

2.  다음 옵션 중 하나를 선택 합니다.

    -   **다음 위치에서 Windows 용 디버깅 도구를 사용**합니다. 이 옵션을 선택 하면 도구의 위치로 이동할 수 있습니다.

    -   **복구 하려는 시스템에서 Windows 용 디버깅 도구를 찾습니다**. 이 옵션을 선택 하면 Windows 용 디버깅 도구가 문제 컴퓨터에 없는 경우 **충돌 분석기** 가 작동 하지 않습니다.

3.  완료 했으면 **다음**을 클릭 합니다.

### DaRT 복구 이미지에 독립 실행형 시스템 Sweeper 대 한 정의를 추가 하려면

정의는 알려진 맬웨어 및 기타 잠재적으로 원치 않는 소프트웨어의 리포지토리입니다. 맬웨어는 지속적으로 개발 되 고 있기 때문에 **독립 실행형 시스템 Sweeper** 는 현재 정의에 따라 컴퓨터에서 설치, 실행 또는 설정을 변경 하려고 하는 소프트웨어가 잠재적으로 원치 않거나 악의적인 소프트웨어 인지 여부를 결정 합니다.

DaRT 복구 이미지에 최신 정의를 포함 하려면 (권장) **예, 최신 정의를 다운로드 하세요.** 정의 업데이트가 자동으로 시작 됩니다. 이 과정을 완료 하려면 인터넷에 연결 되어 있어야 합니다.

정의 업데이트를 건너뛰려면 **아니요를 클릭 하 고 나중에 수동으로 정의를 다운로드**합니다. 정의는 DaRT 복구 이미지에 포함 되지 않습니다.

복구 이미지에 최신 정의를 포함 하지 않기로 결정 하거나, **독립 실행형 시스템 Sweeper**를 사용할 준비가 된 시점에 복구 이미지에 포함 된 정의가 더 이상 유효 하지 않은 경우에는 **독립 실행형 시스템 Sweeper**에서 제공 하는 지침에 따라 검색을 시작 하기 전에 최신 정의를 구하십시오.

**중요**  정의가 없는 경우 스캔할 수 없습니다.

 

완료 했으면 **다음**을 클릭 합니다.

### DaRT 복구 이미지에 드라이버 추가

**주의**  사항 기본적으로 DaRT 복구 이미지에 드라이버를 추가 하면 해당 폴더에 있는 모든 추가 파일 및 하위 폴더가 복구 이미지에 추가 됩니다. 자세한 내용은 [DaRT 7.0 문제 해결](troubleshooting-dart-70-new-ia.md)을 참조 하세요.

 

컴퓨터를 복구할 때 필요할 수 있는 DaRT7의 복구 이미지에 추가 드라이버를 포함 해야 합니다. 여기에는 일반적으로 Windows DVD에 포함 되지 않은 저장소 또는 네트워크 컨트롤러가 포함 될 수 있습니다.

**중요**  포함할 드라이버를 선택할 때 무선 연결이 DaRT에서 지원 되지 않는다는 점에 유의 하세요 (예: Bluetooth 또는 802.11 a/b/g/n).

 

**복구 이미지에 저장소 또는 네트워크 컨트롤러 드라이버를 추가 하려면**

1.  **DaRT 복구 이미지 마법사**의 **추가 드라이버** 대화 상자에서 **장치 추가**를 클릭 합니다.

2.  드라이버에 추가할 파일을 찾은 다음 **열기**를 클릭 합니다.

    **참고**  **드라이버** 파일은 저장소 또는 네트워크 컨트롤러의 제조업체에서 제공 합니다.

     

3.  포함 하려는 모든 드라이버에 대해 1 ~ 2 단계를 반복 합니다.

4.  완료 했으면 **다음**을 클릭 합니다.

### DaRT 복구 이미지에 파일을 추가 하려면

복구 이미지에 파일을 추가 하 여 컴퓨터 문제를 진단 하는 데 사용할 수 있도록 하려면 다음 단계를 따르세요.

1.  **DaRT 복구 이미지 마법사**의 **추가 파일** 대화 상자에서 **파일 표시**를 클릭 합니다. 이렇게 하면 공유 파일을 보유 하 고 있는 폴더가 표시 되는 탐색기 창이 열립니다.

2.  대화 상자에 나열 된 폴더에 하위 폴더를 만듭니다.

3.  원하는 파일을 새 하위 폴더에 복사 합니다.

4.  완료 했으면 다음을 클릭 **합니다.**

### DaRT 복구 이미지가 포함 된 ISO의 위치를 선택 하려면

ISO 이미지를 만들 위치를 지정 하려면 다음 단계를 따르세요.

1.  **DaRT 복구 이미지 마법사**의 **시작 이미지 만들기** 대화 상자에서 **찾아보기를**클릭 합니다.

2.  다른 **이름으로 저장** 창에서 원하는 위치를 찾은 다음 **저장**을 클릭 합니다.

3.  완료 했으면 **다음**을 클릭 합니다.

ISO 이미지의 크기는 사용자가 선택 하는 도구와 마법사에서 추가 하는 파일에 따라 달라 집니다.

CD 또는 DVD를 굽는 대부분의 프로그램에는 해당 확장이 필요 하기 때문에 마법사는 ISO 이미지에 .iso 파일 확장명을 사용 해야 합니다 **.** 다른 위치를 지정 하지 않으면 이름 **DaRT70**와 함께 바탕 화면에 iso 이미지가 만들어집니다.

### CD 또는 DVD에 복구 이미지를 구우려면

**DaRT 복구 이미지 마법사** 가 컴퓨터에서 호환 되는 cd-rw 드라이브를 감지 하면 ISO 이미지를 디스크에 굽는 것을 제공 합니다. CD 또는 DVD를 구울 때 마법사에서 드라이브를 인식 하지 못하는 경우 드라이브에 포함 된 프로그램과 같은 다른 프로그램을 사용 해야 합니다. Duplicator, 복제 서비스 또는 CD 또는 DVD 굽기 소프트웨어를 사용 하 여 추가 복사본을 만들 수 있습니다.

1.  **DaRT 복구 이미지 마법사**의 **기록 가능한 CD/dvd로 굽기** 대화 상자에서 **기록 가능한 다음 CD/dvd 드라이브로 이미지 굽기를**선택 합니다.

2.  CD 또는 DVD 드라이브를 선택 합니다.

    **참고**  드라이브가 인식 되지 않고 새 드라이브를 설치 하는 경우 **드라이브 목록 새로 고침** 을 클릭 하 여 마법사가 사용 가능한 드라이브 목록을 강제로 업데이트 하도록 할 수 있습니다.

     

3.  **다음**을 클릭합니다.
