# MED-V 작업 영역에 성능 설정을 적용하는 방법


각 MED-V 작업 영역에 대해 다음 성능 설정을 정의할 수 있습니다.

VM 메모리 할당

성능 설정은 **정책** 모듈의 **성능** 탭에서 구성 합니다.

**MED-V 작업 영역에 성능 설정을 적용 하려면**

1.  MED-V 작업 영역을 클릭 하 여 구성 합니다.

2.  **성능** 창에서 다음 표에 설명 된 대로 설정을 구성 합니다.

3.  **정책** 메뉴에서 **커밋을**선택 합니다.
