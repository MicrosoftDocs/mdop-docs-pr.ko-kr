# 서버 관리 콘솔의 응용 프로그램 결과 창


**응용 프로그램 결과** 창에 사용 가능한 응용 프로그램 목록이 표시 됩니다.

**결과** 창의 아무 곳 이나 마우스 오른쪽 단추로 클릭 하 여 응용 프로그램 또는 응용 프로그램 그룹을 제외한 다음 요소를 포함 하는 팝업 메뉴를 표시 합니다.

<a href="" id="refresh"></a>**새로 고침**  
응용 프로그램 목록을 새로 고칩니다.

<a href="" id="export-list"></a>**목록 내보내기**  
**결과** 창의 내용이 들어 있는 탭으로 구분 된 텍스트 파일을 만듭니다. 이 항목에는 만드는 텍스트 파일의 위치를 지정 하는 표준 **파일 저장** 대화 상자가 표시 됩니다. **목록 내보내기** 기능에 대 한 자세한 내용은 Microsoft 관리 콘솔에 대 한 설명서를 참조 하세요.

<a href="" id="view"></a>**보기**  
**결과** 창의 모양 및 콘텐츠를 변경 합니다.

<a href="" id="arrange-line-up-icons"></a>**정렬/줄 위로 아이콘**  
**결과** 창에서 아이콘을 구성 합니다.

<a href="" id="help"></a>**Help**  
Application Virtualization Management Console에 대 한 도움말 시스템을 표시 합니다.

**결과** 창에서 응용 프로그램을 마우스 오른쪽 단추로 클릭 하 여 다음 요소를 포함 하는 팝업 메뉴를 표시 합니다.

<a href="" id="move"></a>**Move**  
응용 프로그램 그룹으로 응용 프로그램을 이동 합니다.

<a href="" id="copy"></a>**Copy**  
응용 프로그램을 다른 응용 프로그램 가상화 시스템에 복사 합니다.

<a href="" id="duplicate"></a>**복제**  
**결과** 창에서 응용 프로그램을 복제 합니다.

<a href="" id="delete"></a>**삭제**  
응용 프로그램 가상화 시스템에서 응용 프로그램을 제거 합니다.

<a href="" id="rename"></a>**Rename**  
응용 프로그램의 이름을 변경할 수 있습니다.

<a href="" id="properties"></a>**특성**  
선택한 응용 프로그램의 **속성** 대화 상자를 표시 합니다. 이 대화 상자에는 다음과 같은 탭이 있습니다.

-   **일반** 탭-응용 프로그램 아이콘, 응용 프로그램 이름, 패키지 이름을 표시 합니다. 또한이 탭에는 변경할 수 있는 응용 프로그램에 대 한 다음과 같은 특정 정보가 표시 됩니다.

    -   **버전**-적절 한 버전 번호를 입력할 수 있습니다. **사용** 확인란을 선택 하 여 버전 번호를 사용 하도록 설정 합니다.

    -   **설명**-응용 프로그램에 대 한 간략 한 설명을 입력할 수 있습니다.

    -   **Osd 경로**-APPROPRIATED on Osd (개방형 소프트웨어 설명자) 파일의 위치를 입력 하거나 찾아볼 수 있습니다.

    -   **아이콘 경로**-응용 프로그램과 연결 하려는 아이콘 파일의 위치를 입력 하거나 찾아볼 수 있습니다.

    -   **응용 프로그램 라이선스 그룹**-라이선스 그룹의 드롭다운 목록에서 라이선스 그룹을 선택할 수 있습니다.

    -   **서버 그룹**-서버 그룹의 드롭다운 목록에서 서버 그룹을 선택할 수 있습니다.

-   **바로 가기** 탭-바로 가기가 게시 된 위치에 해당 하는 확인란을 표시 합니다. 이 탭에서 확인란을 선택 하거나 선택을 취소할 수 있습니다.

-   **파일 형식 연결** 탭-선택한 응용 프로그램과 연결 된 파일 형식 목록을 표시 합니다. 이 탭에서 파일 형식 연결을 추가, 편집 또는 삭제할 수 있습니다.

-   **액세스 권한** 탭-선택한 응용 프로그램에 대 한 액세스 권한이 있는 그룹 목록을 표시 합니다. 이 탭에서 그룹을 추가, 편집 또는 삭제할 수 있습니다.

<a href="" id="help"></a>**Help**  
Application Virtualization Management Console에 대 한 도움말 시스템을 표시 합니다.

응용 프로그램 그룹을 마우스 오른쪽 단추로 클릭 하 여 다음 요소를 포함 하는 팝업 메뉴를 표시 합니다.

<a href="" id="new-application-group"></a>**새 응용 프로그램 그룹**  
새 응용 프로그램 그룹 마법사를 표시 합니다. 해당 필드에 새 응용 프로그램 그룹의 이름을 추가 하 고 **마침을**클릭 합니다.

<a href="" id="new-application"></a>**새 응용 프로그램**  
새 응용 프로그램 마법사를 표시 합니다. 마법사를 탐색 하 여 응용 프로그램을 추가 합니다.

<a href="" id="import-applications"></a>**응용 프로그램 가져오기**  
기존 응용 프로그램을 응용 프로그램 가상화 관리 콘솔로 가져오는 데 사용할 수 있는 찾아보기 대화 상자를 표시 합니다. OSD 파일 또는 SPRJ (Sequencer 프로젝트) 파일을 가져올 수 있습니다.

<a href="" id="move"></a>**Move**  
응용 프로그램을 응용 프로그램 그룹으로 이동 합니다.

<a href="" id="copy"></a>**Copy**  
응용 프로그램 그룹을 새 서버에 복사 합니다.

<a href="" id="new-window-from-here"></a>**여기에서 새 창**  
선택한 노드를 루트 노드로 하 여 새 관리 콘솔을 엽니다.

<a href="" id="delete"></a>**삭제**  
서버에서 응용 프로그램 그룹을 제거 합니다.

<a href="" id="rename"></a>**Rename**  
응용 프로그램 그룹의 이름을 변경 합니다.

<a href="" id="refresh"></a>**새로 고침**  
응용 프로그램 그룹을 새로 고칩니다. **결과** 창에 전체 응용 프로그램 노드가 표시 되는 경우 창이 전환 되어 응용 프로그램 그룹의 내용이 표시 됩니다.

<a href="" id="properties"></a>**특성**  
선택한 응용 프로그램 그룹에 대 한 **속성** 대화 상자를 표시 합니다. 이 대화 상자에는 다음과 같은 탭이 있습니다.

-   **일반** 탭-응용 프로그램 그룹 아이콘 및 응용 프로그램 그룹 이름을 표시 합니다. 또한이 탭에는 변경할 수 있는 응용 프로그램 그룹에 대 한 제한 된 정보만 표시 됩니다.

    -   **버전**-응용 프로그램 그룹의 버전 번호를 입력할 수 있습니다.

    -   **설명**-응용 프로그램 그룹에 대 한 간략 한 설명을 입력할 수 있습니다.

    -   **Osd 경로**-osd 파일이 있는 위치를 입력 하거나 찾아볼 수 있습니다.

    -   **아이콘 경로**-아이콘 파일이 있는 위치를 입력 하거나 찾아볼 수 있습니다.

    -   **응용 프로그램 라이선스 그룹**-라이선스 그룹의 드롭다운 목록에서 라이선스 그룹을 선택할 수 있습니다.

    -   **서버 그룹**-서버 그룹의 드롭다운 목록에서 서버 그룹을 선택할 수 있습니다.

-   **바로 가기** 탭-바로 가기가 게시 된 위치에 해당 하는 확인란을 표시 합니다. 이 탭에서 확인란을 선택 하거나 선택을 취소할 수 있습니다.

-   **파일 연결** 탭-파일 형식 연결 목록을 표시 합니다. 이 탭에서 파일 형식 연결을 추가, 편집 또는 삭제할 수 있습니다.

-   **액세스 권한** 탭-선택한 응용 프로그램 그룹에 대 한 액세스 권한이 있는 그룹 목록을 표시 합니다. 이 탭에서 그룹을 추가, 편집 또는 삭제할 수 있습니다.

<a href="" id="help"></a>**Help**  
Application Virtualization Management Console에 대 한 도움말 시스템을 표시 합니다.
