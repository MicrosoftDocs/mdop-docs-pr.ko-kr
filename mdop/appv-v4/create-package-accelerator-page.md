# 패키지 가속기 만들기 페이지


패키지 가속기 **만들기** 페이지를 사용 하 여 패키지 가속기를 저장할 위치를 지정 합니다.

이 페이지에는 다음 요소가 포함 되어 있습니다.

<a href="" id="browse"></a>**찾아보기**  
**찾아보기를** 클릭 하 여 패키지 가속기를 저장할 위치를 지정 합니다.
