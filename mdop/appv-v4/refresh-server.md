# REFRESH SERVER


이 명령은 서버에서 게시 정보를 업데이트 합니다.

`SFTMIME REFRESH SERVER:server-name [/LOG log-pathname | /CONSOLE | /GUI]`

<td align="left"><p>서버: &lt; 서버 이름&gt;</p></td>
