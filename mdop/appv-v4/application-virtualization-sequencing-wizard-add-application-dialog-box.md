# Application Virtualization 시퀀싱 마법사 추가 응용 프로그램 대화 상자


**응용 프로그램 추가** 대화 상자를 사용 하 여 시퀀싱을 위해 시퀀싱 된 응용 프로그램 패키지에 응용 프로그램을 추가할 수 있습니다. 이 대화 상자에는 다음 요소가 포함 되어 있습니다.

<a href="" id="application-path"></a>**응용 프로그램 경로**  
응용 프로그램의 경로를 지정 합니다.

<a href="" id="name"></a>**이름**  
OSD 파일에 나열 된 응용 프로그램의 이름을 지정 합니다.

<a href="" id="version"></a>**버전**  
OSD 파일에 나열 된 응용 프로그램 버전을 지정 합니다.

<a href="" id="osd-file-name"></a>**OSD 파일 이름**  
응용 프로그램 바로 가기가 응용 프로그램을 열도록 하는 OSD 파일의 이름을 지정 합니다.
