# 응용 프로그램 가상화 시퀀싱 마법사 응용 프로그램 시작 페이지


앱-V 시퀀싱 마법사의 **응용 프로그램 시작** 페이지를 사용 하 여 Sequencer가 스트리밍을 위해 패키지를 최적화할 수 있도록 응용 프로그램을 실행 하 고 응용 프로그램을 처음에 대상 컴퓨터에서 시작 하는 방법을 구성할 수 있습니다. 이 페이지에는 다음 표에 설명 된 요소가 포함 되어 있습니다.
