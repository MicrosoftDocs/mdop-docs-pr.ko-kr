# 관리자 그룹을 삭제하는 방법


응용 프로그램 가상화 서버 관리 콘솔의 **관리자** 노드에서 다음 절차를 사용 하 여 관리자 그룹을 삭제할 수 있습니다.

**관리자 그룹을 삭제 하려면**

1.  **관리자** 노드를 클릭 하 여 **결과** 창에 관리자 그룹의 목록을 표시 합니다.

2.  **작업** 창 (맨 오른쪽 창)에서 **삭제**를 클릭 합니다.

3.  **예**를 클릭합니다.
