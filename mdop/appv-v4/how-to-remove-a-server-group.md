# 서버 그룹을 제거하는 방법


응용 프로그램 가상화 서버 관리 콘솔에서 다음 절차를 사용 하 여 응용 프로그램 가상화 시스템에서 그룹의 모든 서버를 포함 하는 서버 그룹을 제거할 수 있습니다.

**서버 그룹을 제거 하려면**

1.  왼쪽 창에서 **서버 그룹** 노드를 클릭 하 여 서버 그룹 목록을 확장 합니다.

2.  원하는 서버 그룹을 마우스 오른쪽 단추로 클릭 하 고 **삭제**를 선택 합니다.

3.  메시지가 표시 되 면 **예**를 클릭 합니다.
