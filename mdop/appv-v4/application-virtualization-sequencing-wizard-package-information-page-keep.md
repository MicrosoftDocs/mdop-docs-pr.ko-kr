# 응용 프로그램 가상화 시퀀싱 마법사 패키지 정보 페이지


Application Virtualization (App-v) 마법사의 **패키지 정보** 페이지를 사용 하 여 시퀀싱 하는 응용 프로그램에 연결 되는 정보를 지정 합니다. 이 페이지에는 다음 요소가 포함 되어 있습니다.
