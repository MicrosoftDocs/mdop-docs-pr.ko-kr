# 다시 시작 작업 실패 대화 상자(App-V 4.6 SP1)


순서를 시도 하는 응용 프로그램과 연결 된 다시 시작 작업이 실패 했습니다.

설치를 완료 한 다음 Microsoft Application Virtualization (App-v) Sequencer를 실행 하는 컴퓨터를 수동으로 다시 시작 합니다.
