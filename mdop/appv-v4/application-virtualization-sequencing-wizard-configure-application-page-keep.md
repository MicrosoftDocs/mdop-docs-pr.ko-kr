# 응용 프로그램 가상화 시퀀싱 마법사 응용 프로그램 구성 페이지


앱-V 시퀀싱 마법사의 **응용 프로그램 구성** 페이지를 사용 하 여 시퀀싱 된 응용 프로그램 패키지에 응용 프로그램을 추가 하 고 바로 가기 및 파일 형식 연결을 추가 하거나 삭제 합니다. 이 페이지에는 다음 표에 설명 된 요소가 포함 되어 있습니다.
