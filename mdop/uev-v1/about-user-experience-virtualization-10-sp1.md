# User Experience Virtualization 1.0 SP1 정보


Microsoft UE-V (User Experience Virtualization) 1.0 서비스 팩 1에서는 버전을 1.0.414에서 1.0.520로 변경 합니다. UE-V Agent setup.exe 또는 UE-V 생성기 setup.exe 시작 되 면 업그레이드의 필요성을 감지 하 고 UE-V Agent 또는 생성기를 업그레이드 합니다.

## 이제 추가 언어가 지원 됩니다.

## Office 2007 설정 위치 템플릿


UE-V 에이전트 설치 소프트웨어는 에이전트를 설치 하 고 공통 Microsoft 응용 프로그램에 대 한 기본 설정 위치 템플릿 그룹을 등록 합니다. Microsoft Office 2007는 이제 이러한 응용 프로그램의 일부입니다. Office 2007 서식 파일에는 MicrosoftOffice2007.xml 및 MicrosoftCommunicator2007.xml 두 가지가 있습니다. 이러한 설정 위치 템플릿은 다음 응용 프로그램에 대해 Microsoft Office 2007에서 설정을 캡처합니다.

-   Microsoft Access 2007

-   Microsoft Communicator 2007

-   Microsoft Excel 2007

-   Microsoft InfoPath 2007

-   Microsoft OneNote 2007

-   Microsoft Outlook 2007

-   Microsoft PowerPoint 2007

-   Microsoft Project 2007

-   Microsoft Publisher 2007

-   Microsoft SharePoint Designer 2007

-   Microsoft Visio 2007

-   Microsoft Word 2007

### Office 2010 설정 위치 템플릿 업데이트

설정 위치 서식 파일에 대 한 업데이트도 적용 되었습니다. 해당 변경 내용은 다음과 같습니다.

-   Office 2010 템플릿에 새 서식 파일을 추가 하 여 Microsoft SharePoint Designer 2010에 대 한 지원이 추가 되었습니다 (MicrosoftOffice2010Win32.xml 및 MicrosoftOffice2010Win64.xml).

-   상태 표시줄 사용자 지정-Word, Excel, PowerPoint를 비롯 한 사소한 버그 수정

## 카탈로그 업데이트에 대 한 예약 된 작업이 이제 임의 상태입니다.


서식 파일 자동 업데이트 작업은 설정 템플릿 카탈로그에서 새 서식 파일, 업데이트 또는 제거 되었는지 확인 합니다. 이 작업은 SettingsTemplateCatalog가 구성 된 경우에만 실행 됩니다. 서식 파일 자동 업데이트 작업은 UE-V SP1이 설치 된 디렉터리에 위치 하는 ApplySettingsCatalog.exe 파일을 실행 하 고 1 시간 동안 업데이트를 임의화 하도록 UE-V sp1(가 변경 되었습니다.

## Citrix EdgeSight에 대 한 지원


Citrix EdgeSight를 사용 하는 서버에서 UE-V를 실행 하는 중에 충돌이 발견 되었습니다. UE-V 1.0 SP1이이 문제를 해결 합니다.

## Internet Explorer 즐겨찾기 인덱싱


UE-V를 사용 하 여 한 컴퓨터에서 다른 컴퓨터로 Internet Explorer 즐겨찾기를 로밍 하면 동기화 된 컴퓨터의 주소 표시줄에 있는 즐겨찾기 주소 인덱싱이 이제 업데이트 됩니다. 사용자가 주소 표시줄에 입력 하면 roamed 즐겨찾기가 동기화 된 컴퓨터에서 사용 가능한 검색 결과로 표시 됩니다.

## UE-V 에이전트 및 UE-V 생성기에 대 한 새로운 setup.exe 명령줄 매개 변수


UE-V 1.0 SP1 릴리스를 사용 하면 다음과 같은 명령줄 매개 변수를 추가로 허용 하도록 UE-V Generator와 UE-V 생성기에 대 한 setup.exe 업데이트 되었습니다.

1.  `CEIPENABLED` -Microsoft 사용자 환경 개선 프로그램에 포함 되는 옵션을 설정에 적용할 수 있습니다.

2.  `INSTALLFOLDER` -에이전트 또는 생성기에 대해 다른 설치 폴더를 설정할 수 있습니다.

3.  `MUENABLED` -설치 프로그램이 Microsoft Update 프로그램에 포함 된 옵션을 허용 하도록 허용 합니다.
