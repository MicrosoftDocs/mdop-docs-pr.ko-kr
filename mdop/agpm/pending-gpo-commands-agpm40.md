# 보류 중인 GPO 명령


**보류 중** 탭:

-   GPO (그룹 정책 개체)에 대 한 요청을 표시 합니다 (예: 만들기, 제어, 배포 또는 삭제).

-   보류 중인 요청에 응답 하 고 Gpo의 기록 및 보고서를 표시 하는 명령이 포함 된 바로 가기 메뉴를 제공 합니다.

-   선택한 GPO에 대 한 액세스 권한이 있는 그룹 및 사용자의 목록을 표시 합니다.

이 탭에서 **그룹 정책 개체** 목록을 마우스 오른쪽 단추로 클릭 하면 다음 중 해당 되는 옵션을 비롯 한 바로 가기 메뉴가 표시 됩니다.
